Build Optimizations

According to Web Almanac, images account for a huge portion of the typical website's page weight and can have a sizable impact on your website's LCP performance. The Next.js Image componet extends the HTML image element with features for automatic image optimization.

  1. Size optimization - Automatically serve correctly sized mages for each device, using modern image formats like WebP and AVIF
  2. Visual Stability - Prevent layout shift automatically when images are loading
  3. Faster Page Loads - Images are only loaded when they enter the viewport using native browser lazy loading, with optional blue-up placeholders
  4. Asset Flexibility - On-demonad image resizing, even for images stored on remote servers
